## 📌 Project Overview
This project is a modern, responsive personal portfolio website designed to showcase skills, projects, resume, and contact information. Built with semantic HTML, custom CSS styling, and JavaScript for interactive elements.

---

## 🛠️ Tech Stack & Technologies Used
* **HTML5:** Semantic layout and structure.
* **CSS3:** Custom styling, layout design, and responsive media queries.
* **JavaScript:** Interactivity and dynamic client-side logic.
* **GitHub Pages:** Deployment and live web hosting.
